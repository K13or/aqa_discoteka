# Android VPN App

Это стартовый каркас Android-приложения для VPN-сервиса.

## Что уже есть

- `VpnService`-база для системной интеграции
- простая Compose-UI для connect/disconnect сценария
- изолированная доменная логика в `vpn/`
- профили VPN и `ViewModel` для управления UI-состоянием
- импорт и парсинг `VLESS` subscription
- unit-тесты для валидации конфигурации и orchestration
- unit-тесты для профилей и `ViewModel`
- unit-тесты для импорта профилей

## Что важно

Сейчас это не полноценная реализация VPN-протокола. Каркас поднимает Android `VpnService` и тестируемую оболочку вокруг него, но реальный туннельный backend, обмен ключами, авторизация, telemetry и защита конфигов ещё не реализованы.

## Замечание по текущему профилю

Проверенная subscription-ссылка сейчас возвращает `VLESS`-записи с адресом `0.0.0.0:1` и служебными сообщениями о неподдерживаемом приложении. Такие записи добавленный импортёр помечает как невалидные и не импортирует.
